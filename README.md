# Password Generator (CLI)

## Description
A Python application that generates secure random passwords.

## Features
- Random password generation
- Uppercase letters
- Lowercase letters
- Numbers
- Special characters
- User-defined password length

## Technologies Used
- Python 3
- random
- string

## How to Run

```bash
python password_generator.py
```
